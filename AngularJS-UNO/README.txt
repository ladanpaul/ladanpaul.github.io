Начало игры:
  - Пользователь в форме вводит имена игроков.
  - Пользователь выбирает счёт до которого идёт игра.
Во время игры:
  - Пользователь может добавить\удалить игрока.
  - Пользователь может остановить игру.
  - Пользователь может начать новую игрую.
  - Пользователь пишет счёт каждому игроку который сумируется к текущему, при нажатии на кнопку +
Конец игры:
  Тот пользователь у которого счёт = счёту который был указан при старте - проиграл.
